Про застосунок

Це простий веб-застосунок на Go, який демонструє основні концепції роботи з Docker

Застосунок складається з Go коду, HTML шаблонів та CSS стилів

Можете змінювати будь-які файли та перебудовувати Docker образ

Що варто реалізувати

  • Використати multi-stage builds для зменшення розміру образу
  • Запускати від імені non-root користувача для безпеки
  • Використати кешування шарів правильно впорядковуючи команди
  • Використати конкретні версії базових образів (уникати 'latest')
  • Мінімізувати кількість шарів
  • Використати .dockerignore для виключення зайвих файлів

Ідеї для експериментів

  1. Змініть цей текст і перебудуйте образ - зверніть увагу які кроки кешуються
  2. Спробуйте змінити версію залежності в go.mod
  3. Порівняйте розміри образів з multi-stage builds та без них
  4. Перевірте, від імені якого користувача працює застосунок всередині контейнера

Успіхів!
